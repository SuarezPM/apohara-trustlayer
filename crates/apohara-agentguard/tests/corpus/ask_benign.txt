# apohara-agentguard Ask corpus — BENIGN
#
# 30+ commands that MUST be classified as Allow (or Block) — NEVER
# Ask — under the Story 6 ask-corpus policy. The policy uses
# `budgets.per_tool.Bash.max_invocations = 0` to escalate every
# Bash call to Ask, so the BENIGN corpus contains ONLY non-Bash
# commands (Read, Write, Edit, WebFetch, UserPromptSubmit) that
# the gate / pathguard / firewall also wouldn't Block for a
# non-secret path / non-injection URL.
#
# Lines starting with `#` and blank lines are ignored by the loader.
